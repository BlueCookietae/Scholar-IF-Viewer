🎓 Scholar IF Viewer(JCR Impact Factor & Top % Display)Scholar IF Viewer는 구글 스칼라(Google Scholar)의 검색 결과와 연구자 프로필 페이지에서 저널의 **임팩트 팩터(Impact Factor)**와 분야 내 상위 백분율(Top %) 정보를 실시간으로 보여주는 크롬 확장 프로그램입니다.연구자들이 논문의 신뢰도와 영향력을 즉각적으로 판단할 수 있도록 돕기 위해 제작되었습니다.✨ 주요 기능실시간 IF 표시검색 결과 및 프로필 페이지의 저널 이름 옆에 강렬한 빨간색으로 Impact Factor를 표시합니다.Q1 저널 하이라이트 (Top N%)해당 분야 상위 25% 이내인 Q1 저널의 경우, 황금색 뱃지와 함께 **구체적인 상위 %**를 소수점 단위로 보여줍니다. (예: TOP 0.4%)강력한 저널 매칭콜론(:), 하이픈(-), 마침표(.) 등 기호 차이나 약어 표기(Abbreviation)에 상관없이 정확하게 저널 데이터를 찾아냅니다.로컬 실행외부 API를 호출하지 않고 내장된 데이터를 사용하므로 속도가 매우 빠르고 개인정보 유출 우려가 없습니다.🛠 설치 방법 (개발자 모드)현재 이 확장 프로그램은 오픈 소스로 제공되며, 크롬의 '개발자 모드'를 통해 수동으로 설치할 수 있습니다.이 저장소를 복제(Clone)하거나 **[Download ZIP]**을 눌러 압축을 풉니다.크롬 브라우저 주소창에 chrome://extensions/를 입력합니다.우측 상단의 [개발자 모드] 스위치를 켭니다.왼쪽 상단의 **[압축해제된 확장 프로그램을 로드합니다]**를 클릭합니다.본 저장소의 extension 폴더를 선택합니다.구글 스칼라에 접속하여 논문을 검색해 보세요!📊 데이터 업데이트 방법직접 수집한 JCR 데이터를 사용하고 싶다면 아래 단계를 따르세요.JCRImpactFactors2025.xlsx 파일(혹은 최신 연도 파일)을 준비합니다.제공된 converter.py를 실행하여 새로운 data.json을 생성합니다.python converter.py
